## 简介：
拉取期货数据保存到本地，大模型分析当前中国的经济形势，输出结果在控制台

## 使用说明
用akshare拉取期货，存储在mysql数据库中，然后根据不同品种导入到不同表中，分析表中的数据获取指标，再用LMStudio的服务分析期货数据以下是分析结果
使用该项目需要按照mysql server和lm studio，
mysql server端口应存放在3306，lm studio端口设置于8085
配置好LM Studio 和 mysql server后，安装项目根目录下的requirements.txt所需依赖
pip install -r requirements.txt
第一次使用运行initialize.py进行初始化，拉取数据并建立数据库表格
日常使用获取周报，使用main.py

### 步骤6：最终总结报告

**【当前经济形势一句话核心判断】**
宏观环境呈现典型的**“金融端定价复苏、实体端需求滞后”**的非均衡特征，市场正处于由流动性宽松与外需韧性支撑的“估值修复期”，但内需（地产/制造业）的结构性疲软构成了主要的下行压制。

---

#### 一、 三大关键驱动因素
1.  **长端利率下行与避险逻辑共振**：根据国债26年合约强势上涨（价格涨/收益率降）与贵金属（白银、黄金远期）强势上涨的趋势，市场正在定价一个长期低利率环境及潜在的抗通胀/避险需求。
2.  **外需韧性对冲内需疲软**：根据集运指数近期上涨与黑色系（螺纹钢、焦煤等）持续弱势下跌的背离，当前经济增长动能主要依赖全球贸易支撑，而非国内基建或地产驱动。
3.  **工业需求品种分化**：根据有色金属中铜、锡表现强势而锌、镍相对疲软的趋势，宏观复苏并非全面爆发，而是集中在电力、半导体等特定高景气制造业领域。
